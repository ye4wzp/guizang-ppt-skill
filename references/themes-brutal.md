# 新丑风格 · 主题色预设（Brutalism Themes）

4 套基于新丑主义（Brutalism / Anti-Design）的主题色板。**每套都遵循"超粗无衬线 + 粗描边 + 色块装饰 + 故意打破网格"的视觉语言**——大胆、粗粝、反精致。

---

## 使用方法

1. 问用户选哪套（或基于内容推荐一套）
2. 打开 `assets/template-brutal.html` 的 `<style>` 块
3. 找到开头的 `:root{` 块
4. **整体替换**标有"主题色"注释的所有变量：`--bg` / `--fg` / `--red` / `--yellow` / `--blue`
5. 其他 CSS 都走 `var(--...)`，无需任何其他改动

---

## ⚡ 经典 Classic（默认）

**适合**：音乐节、创意发布会、设计分享、独立杂志、反传统场景。
**调性**：纯白底 + 三原色（红/黄/蓝），David Carson 式反叛。

```css
--bg:#ffffff;
--fg:#000000;
--red:#ff3b30;
--yellow:#ffcc00;
--blue:#007aff;
```

**使用要点**：
- 三原色同时出现是经典新丑标志，不要削减到只剩一个
- 黄色做贴纸/标签最醒目，红色做警告/强调，蓝色做辅助
- 深色 slide（`--fg` 做背景）上装饰块用 `--yellow` 对比最强

---

## ⚫ 单色 Mono

**适合**：极简主义、建筑、排版实验、不想用彩色的场景。
**调性**：纯黑白 + 灰阶，像 Xerox 复印机印出来的独立杂志。

```css
--bg:#ffffff;
--fg:#000000;
--red:#888888;
--yellow:#cccccc;
--blue:#aaaaaa;
```

**使用要点**：
- 所有 accent 退化为灰阶，视觉冲击靠字号对比和布局张力
- 适合文字量大、排版本身就是主角的 deck
- 深色 slide 上的灰色装饰块需要保持足够对比度

---

## 🎨 莫兰迪 Morandi

**适合**：生活方式、女性、美妆、花艺、咖啡、温柔但有态度的分享。
**调性**：暖米底 + 低饱和粉彩，像莫兰迪的静物画，新丑的温柔变体。

```css
--bg:#f5f0eb;
--fg:#3d3632;
--red:#c4907a;
--yellow:#d4c4a0;
--blue:#8fa3b0;
```

**使用要点**：
- 底色不再是纯白而是暖米，整体更柔和
- 粗描边仍然保留（3px），但颜色从黑变为深棕 `--fg`
- 贴纸标签用 `--yellow`（沙色）比用 `--red`（玫瑰）更和谐
- 适合"新丑但不想太aggressive"的场景

---

## 🔥 热力 Heat

**适合**：暗黑主题、科技发布会、游戏、电竞、夜店文化、热血分享。
**调性**：纯黑底 + 红→橙→琥珀暖色系，像燃烧的余烬。

```css
--bg:#0a0a0a;
--fg:#f0f0f0;
--red:#e63946;
--yellow:#f4845f;
--blue:#f7b267;
```

**使用要点**：
- 默认深色模式，`--fg` 做文字（近白），`--bg` 做背景（近黑）
- 三个 accent 从红到橙是同色系渐变，不是三原色
- 装饰块在深底上用 `--red` 最有冲击力
- 浅色 slide（`--fg` 做背景）上的深色文字用 `--bg`

## 切换原则

- **一份 deck 只用一套主题**，不要中途换色
- 三个 accent 色在一套主题内可以同时使用（新丑允许）
- 粗描边（`3px solid`）是新丑核心，不要削弱为 1px
- 贴纸标签（`.sticker`）的旋转角度保持 -2° ~ -1°，不要转太多

## ❌ 不要做的事

- ❌ **不允许混搭不同主题的 accent**（例如经典红 + 莫兰迪蓝）——会失去主题统一性
- ❌ **不允许用户自定义 hex 值**——委婉拒绝，展示 4 套预设让选
- ❌ **不要加圆角**——新丑风的所有元素必须直角（`border-radius:0`）
- ❌ **不要加阴影**——新丑风拒绝 `box-shadow`，粗描边就是全部
- ❌ **不要削弱描边**——3px 是底线，不要为了"好看"改成 1px
