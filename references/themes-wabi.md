# 侘寂风格 · 主题色预设（Wabi-Sabi Themes）

5 套基于日式侘寂美学（Wabi-Sabi）的主题色板。**每套都遵循"极细宋体 + 和纸纹理 + 墨渍装饰 + 非对称留白"的视觉语言**——素雅、克制、有呼吸感。

---

## 使用方法

1. 问用户选哪套（或基于内容推荐一套）
2. 打开 `assets/template-wabi.html` 的 `<style>` 块
3. 找到开头的 `:root{` 块
4. **整体替换**标有"主题色"注释的所有变量：`--ink` / `--ink-rgb` / `--paper` / `--paper-rgb` / `--washi` / `--sumi` / `--sumi-rgb` / `--accent`
5. 其他 CSS 都走 `var(--...)`，无需任何其他改动

## 🖋 暖墨 Warm Ink（默认）

**适合**：通用分享、行业观察、商业发布、任何场景都安全的默认选择。
**调性**：暖深灰 + 暖米白，Monocle 杂志式侘寂感。

```css
--ink:#2c2c2c;
--ink-rgb:44,44,44;
--paper:#f7f5f0;
--paper-rgb:247,245,240;
--washi:#ece8df;
--sumi:#1a1a1a;
--sumi-rgb:26,26,26;
--accent:#8b7355;
```

## 切换原则

- **一份 deck 只用一套主题**，不要中途换色
- 和纸纹理（`--washi`）自动跟随主题切换，无需额外调整
- 墨渍装饰（`--sumi`）在所有主题下都是最深色，保持一致
- `currentColor` 驱动的 border / icon 会跟随 section 的 text color 自动适配

## ❌ 不要做的事

- ❌ **不允许混搭**（例如 ink 取枯山水的，paper 取朱砂的）——会彻底违和
- ❌ **不允许用户自定义 hex 值**——委婉拒绝，展示 5 套预设让选
- ❌ **不要用纯黑 `#000` 做 ink**——侘寂风的 ink 必须带色调（暖/冷/绿/蓝），纯黑太硬
- ❌ **不要用纯白 `#fff` 做 paper**——侘寂风的 paper 必须带暖调或冷调，纯白太刺眼
